历史

C#语言的历史

C -> 1980s, C++ and Objctive-C -> Mid 1990s, JAVA -> 2000s, 巨硬搞出了C#用来抗衡JAVA的一统天下之势，C#只适用于.NET平台。

.NET平台的历史

Late 1980s, DDE Dynamic Data Exchange -> Early 1990s, OLE -> 1990s, COM, COM+, DCOM -> 2000s, 巨硬搞出了.NET platform
