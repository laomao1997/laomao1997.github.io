#

吉林省疫情发展较为迅速，我有很多朋友、亲人都在吉林，他们的工作、生活、学习无不受到这一波疫情的影响。这一轮疫情不仅影响了吉林人民的生活，还牵动了全国人民的心。我每天关注吉林卫健委通报的数据，揪心之余，想着应该做些什么，于是统计了 3 月 1 日起全部疫情数据，并利用 Python 进行曲线拟合，来预测一下这一波疫情的发展态势。

目前（3 月 24 日）来看，这一轮疫情总体会在4 月初停止新增，预计在 4 月下旬可以解封，其中，吉林市预计会较早完成零新增，预计在 3 月末停止，而长春市，预计要更晚。

# 写在前面，为什么疫情可预测？

通常来讲，传染病的传染速度体现在每日新增感染人数上，而这一数字的每日变化通常遵循正态分布，也就是说，某一特定地区的疫情，其累积确诊病例变化，在控制得当的情况下。应当遵循“S 曲线”，所以只要可以获得一定日子的“每日新增感染人数”数据，就可以利用 Python 进行函数拟合，获得对应的“S 曲线”，来预测疫情的拐点和最大值。

而“S 曲线”的函数表达式为：

y=k1+e(a−bx)y = \frac{k}{1+e^{(a-bx)}} y=​1+e​(a−bx)​​​​k​​

当 y=k2y = \frac{k}{2}y=​2​​k​​时，yyy 增长最快，也就是我们所说的拐点。当 y=ky = ky=k 时，yyy 停止增长，为最大值。

数据来源：吉林省卫健委。 希望疫情早日结束，早日解封，人民生活回归正常。

# 长春市累计确诊病例数预测

import datetime
import numpy as np
from matplotlib import pyplot as plt
from scipy.optimize import curve_fit as curve_fit
from pylab import mpl

# 修改作图字体
mpl.rcParams['font.sans-serif'] = ['Sarasa Gothic SC']
mpl.rcParams['axes.unicode_minus'] = False

# 这里录入每日病例数
totalCasesPerDayArray = [
    0, 0, 0, 5, 12, 19, 36, 48, 71, 73, 136, 967, 1397, 1857, 2901, 3169, 3764, 4507, 5340, 6419, 7856, 9835, 11115, 11691, 12245, 12878, 13584, 14206, 15081, 16078, 17156, 18700, 19423, 19997, 20601, 21418, 22184, 22658, 22808, 22983, 23085, 23166, 23339
]

# 计算上述数据对应天数
daysNum = len(totalCasesPerDayArray)
# 横坐标，表示第几天
xdata = [i+1 for i in range(daysNum)]
# 纵坐标，表示每天病例数
ydata = totalCasesPerDayArray
# 将数据以散点图打印出来
plt.scatter(xdata, ydata, label='data')

# 定义曲线函数


def func(x, k, a, b):
    return k/(1+(k/b-1)*np.exp(-a*x))


# 拟合
popt, pcov = curve_fit(func, xdata, ydata, method='dogbox', bounds=(
    [1000., 0.01, 10.], [10000000., 1.0, 1000.]), maxfev=500000)
k = popt[0]
a = popt[1]
b = popt[2]
print("k = ", k, ", a = ", a, ", b = ", b)
print("k/2 = ", k/2)

# 计算拟合后的数据
additionalDaysNum = 21
x = np.linspace(0, len(xdata) + additionalDaysNum)
y = func(x, *popt)

# 今天
today = datetime.date.today()

# 作图
plt.plot(x, y, color='r', label='fit')
plt.xlabel('Days')
plt.ylabel('Number of Cases')
plt.title('长春市累计确诊病例数预测 ({})'.format(today))
plt.grid()
plt.legend(loc='best')
plt.savefig('D:\\blog\\docs\\.vuepress\\public\\img\\changchun.svg', format="svg")
plt.show()
